一维线性问题

  • 一维 dp[i]定义

    • 长度为 0~i 的数组的最大、最小、一共有 dp[i]个
    • 以 i结尾的数组最大、最小、一共有 dp[i]个（爬楼梯、排列、最大子数组和、递增子序列、单词拆分）
    • 选择 i 的最大、最小、一共有 dp[i]个（爬楼梯、打家劫舍）

  • 二维 dp[i][j]定义

    • 目标为 j 的前 i 个元素大、最小、一共有 dp[i][j]个（找零钱）
    • 长度为 arr[i...j]是否满足条件（回文子序列、回文子串）
    • 在前 i 个 arr[0...i]处于状态（选择）[j]的最大值、最小等（股票、背包）

二维线形问题

  • 二维 dp[i][j]定义
    • 在子数组 arr1[0..i]和子数组 arr2[0..j]中，我们要求的子序列长度为 dp[i][j]。（最长公共子序列）
    • 在前 i 个 arr[0...i]处于状态（选择）[j]的最大值、最小等（股票、背包）
